# `calculator.css` 檔案教學：計算機的「裝潢師傅」

`calculator.css` 檔案是我們計算機網頁的「裝潢師傅」。它負責定義計算機所有看起來的樣子，讓它變得美觀又好用。想像一下，如果你想改變計算機的顏色、字體、按鈕的形狀，或是讓它在不同大小的螢幕上都能漂亮顯示，那麼你就是要來修改這個檔案！

## 檔案結構概覽 (CSS 程式碼)

```css
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    /* 整個網頁的背景、字體、居中設定 */
}

.calculator {
    /* 計算機外框的樣式 */
}

.display {
    /* 顯示螢幕區域的樣式 */
}

#display {
    /* 顯示螢幕輸入框的樣式 */
}

.buttons {
    /* 按鈕排列的樣式 */
}

.btn {
    /* 所有按鈕的共同樣式 */
}

.btn:hover {
    /* 滑鼠移到按鈕上時的樣式 */
}

.btn:active {
    /* 按鈕被點擊時的樣式 */
}

.number {
    /* 數字按鈕的專屬樣式 */
}

.operator {
    /* 運算符號按鈕的專屬樣式 */
}

.clear {
    /* 清除按鈕的專屬樣式 */
}

    /* 等於按鈕的專屬樣式 */
}

.zero {
    /* 數字0按鈕的專屬樣式 */
}

/* 響應式設計：讓網頁在不同裝置上都好看 */

    /* 當螢幕比較小時，調整樣式 */
}

/* 動畫效果 */

    /* 計算機出現時的淡入動畫 */
}

    /* 應用淡入動畫 */

}

    /* 按鈕點擊動畫的細節 */
```

## 主要樣式說明 (白話解釋)

CSS 程式碼由很多「規則」組成，每個規則都告訴瀏覽器某個部分應該長什麼樣子。讓我們看看幾個重要的部分：

### 1. 全局重置 (`*`)：網頁的「大掃除」

*   `margin: 0; padding: 0;`：這兩行就像是給網頁上的所有東西做「大掃除」，把它們預設的外邊距 (margin) 和內邊距 (padding) 都清零。這樣我們就可以從頭開始，精確地控制每個元素的位置。
*   `box-sizing: border-box;`：這是一個很實用的設定！它讓你在設定一個元素的寬度 (width) 和高度 (height) 時，會把內邊距和邊框 (border) 也計算在內。這樣可以避免一些排版上的小麻煩，讓佈局更容易控制。

### 2. `body` 樣式：整個網頁的「背景和中心」

`body` 標籤代表了整個網頁的內容區域。這裡的樣式會影響整個網頁的視覺效果。

*   `font-family`：設定網頁上文字的字體。你可以選擇你喜歡的字體，讓網頁看起來更有風格。
*   `background: linear-gradient(...)`：這行程式碼設定了網頁的背景顏色，而且是一個漂亮的「漸變色」！它讓背景從一種顏色平滑地過渡到另一種顏色，看起來很現代。
*   `min-height: 100vh; display: flex; justify-content: center; align-items: center;`：這幾行是讓計算機在網頁上「居中」顯示的魔法！
    *   `min-height: 100vh;`：讓網頁的最小高度佔滿整個瀏覽器視窗的高度。
    *   `display: flex;`：啟用 Flexbox 佈局，這是一種很方便的排版方式。
    *   `justify-content: center; align-items: center;`：這兩行讓 Flexbox 容器裡面的內容 (也就是我們的計算機) 在水平和垂直方向上都完美居中。這樣無論螢幕多大，計算機都會在正中間。
*   `padding: 20px;`：在網頁內容的四周留出一些空間，避免內容太靠近邊緣，尤其是在小螢幕上會看起來更舒服。

### 3. `.calculator` (計算機容器)：計算機的「外框」

這個 `.calculator` 類別 (class) 對應到 `index.html` 中包裹整個計算機的 `div`。這裡的樣式決定了計算機的整體外觀。

*   `background: rgba(255, 255, 255, 0.1); backdrop-filter: blur(10px);`：這兩行創造了現在很流行的「毛玻璃」效果！它讓計算機的背景有點透明，並且模糊了後面的內容，看起來很有質感。
*   `border-radius: 20px;`：讓計算機的邊框變成圓角，看起來更柔和。
*   `padding: 30px;`：計算機內部內容與邊框之間的空間，讓內容不會擠在一起。
*   `box-shadow: 0 20px 40px rgba(0, 0, 0, 0.3);`：給計算機添加陰影，讓它看起來像是浮在頁面上，更有立體感。
*   `border: 1px solid rgba(255, 255, 255, 0.2);`：一個輕微的白色邊框，讓毛玻璃效果看起來更清晰。
*   `max-width: 350px; width: 100%;`：設定計算機的最大寬度為 350 像素，但如果螢幕更小，它會自動縮小到螢幕的 100% 寬度，這也是響應式設計的一部分。
*   `animation: fadeIn 0.5s ease-out;`：當計算機載入時，會應用一個「淡入」的動畫效果，讓它平滑地出現在螢幕上。

### 4. `.display` 和 `#display` (顯示螢幕)：計算機的「臉」

*   `#display`：這個 ID 對應到 `index.html` 中的顯示螢幕輸入框。這裡的樣式讓顯示螢幕看起來像一個真正的計算機螢幕。
    *   設定了寬度、高度、深色背景、白色文字、字體大小，以及文字靠右對齊。
    *   `border-radius` 和 `box-shadow` 讓顯示螢幕看起來更有設計感。

### 5. `.buttons` (按鈕容器)：按鈕的「排列方式」

*   `display: grid; grid-template-columns: repeat(4, 1fr); gap: 15px;`：這幾行使用了 CSS Grid 佈局，這是一種非常強大的排版工具！
    *   `display: grid;`：啟用 Grid 佈局。
    *   `grid-template-columns: repeat(4, 1fr);`：告訴瀏覽器，把按鈕排成 4 列，每列的寬度都平均分配 (1fr 表示一份空間)。
    *   `gap: 15px;`：設定按鈕之間的間距為 15 像素。

### 6. `.btn` (通用按鈕樣式)：所有按鈕的「共同特徵」

這個 `.btn` 類別包含了所有按鈕都會有的共同樣式。

*   `height: 70px; border: none; border-radius: 15px;`：設定按鈕的高度、移除邊框，並讓按鈕變成圓角。
*   `font-size; font-weight; cursor: pointer;`：設定按鈕上文字的大小、粗細，以及當滑鼠移到按鈕上時，滑鼠指標會變成一個小手 (表示可以點擊)。
*   `transition: all 0.3s ease;`：這是一個很棒的動畫效果！它讓按鈕在樣式改變時 (例如滑鼠移上去或點擊時)，會平滑地過渡，而不是突然改變，讓使用者體驗更好。
*   `box-shadow: 0 5px 15px rgba(0, 0, 0, 0.2);`：給按鈕添加陰影，讓它們看起來更有立體感。
*   `:hover` 和 `:active`：這些是「偽類別」，它們定義了按鈕在特定狀態下的樣式。
    *   `:hover`：當滑鼠移到按鈕上時，按鈕會稍微向上移動 (`transform: translateY(-3px);`) 並改變陰影，給使用者一個「我可以點擊」的提示。
    *   `:active`：當按鈕被點擊時，按鈕會稍微向下移動 (`transform: translateY(1px);`) 並改變陰影，給使用者一個「我被點擊了」的反饋。

### 7. 特定按鈕樣式 (`.number`, `.operator`, `.clear`, `.equals`, `.zero`)：按鈕的「獨特個性」

除了共同樣式外，不同功能的按鈕還有自己獨特的樣式，讓它們更容易區分。

*   這些類別為數字按鈕、運算符號按鈕、清除按鈕、等於按鈕和數字 0 按鈕設定了不同的背景漸變色和文字顏色。
*   `.equals`：使用了 `grid-row: span 2;`，這表示「等於」按鈕會佔用兩行的高度，所以它看起來會比其他按鈕長。
*   `.zero`：使用了 `grid-column: span 2;`，這表示數字 0 按鈕會佔用兩列的寬度，所以它看起來會比其他數字按鈕寬。

### 8. 響應式設計 (`@media`)：讓網頁在「任何裝置」上都好看

*   `@media (max-width: 480px)`：這是一個「媒體查詢」。它的意思是：「當螢幕的寬度小於或等於 480 像素時 (通常是手機螢幕的大小)，請應用這裡面的樣式。」
    *   在這個區塊裡，我們會調整計算機的內邊距、顯示螢幕和按鈕的大小，讓它們在小螢幕上看起來更合適，不會太擁擠。

### 9. 動畫 (`@keyframes`)：讓計算機「動起來」

*   `@keyframes fadeIn`：定義了一個名為 `fadeIn` 的動畫。它讓計算機在載入時，從完全透明且稍微縮小的狀態，平滑地變成完全不透明且正常大小，創造一個淡入的效果。
*   `@keyframes buttonClick`：定義了一個名為 `buttonClick` 的動畫。它讓按鈕在被點擊時，會稍微縮放一下，提供一個視覺上的反饋，讓你知道按鈕被按下了。

## 如何使用這個 `calculator.css` 檔案？

`calculator.css` 檔案是透過 `index.html` 中的 `<link>` 標籤被引入的。當你打開 `index.html` 時，瀏覽器會自動載入並應用這些 CSS 樣式，讓你的計算機呈現出設計好的漂亮外觀。

如果你想修改計算機的顏色、佈局或任何視覺效果，你只需要編輯這個 `calculator.css` 檔案，然後重新整理瀏覽器，就可以看到你的改變了！試著改改看，你會發現網頁設計其實很有趣！
